Functional & Non-functional Requirements

Pose & Vision Core

  • Real-time webcam feed capture at ≥24 FPS
  • MediaPipe Pose: detect 33 body landmarks
  • Joint angle computation (knee, hip, shoulder, spine)
  • Velocity & acceleration tracking per joint
  • Symmetry scoring left/right body halves
  • Range of Motion (ROM) measurement

AI Engine Core

  • Exercise phase segmentation (e.g. squat phases)
  • Phase 1: GRU temporal sequence classifier
  • Phase 2: Transformer + TCN model
  • Error classification: knee valgus, rounded back, incomplete ROM
  • Movement safety scoring per repetition
  • Injury risk flag before error occurs

Feedback UX

  • Visual: skeleton overlay on webcam frame
  • Highlight incorrect joints in real time
  • Text corrections displayed on screen
  • Voice feedback via TTS / Web Speech API
  • Rep counter with quality score per rep
  • Post-session summary report

Patient Dashboard UX

  • Session start / stop with exercise selection
  • Live skeleton + angle visualisation
  • Session history with playback
  • Progress charts (ROM, accuracy over time)
  • Exercise library with reference videos
  • Google OAuth authentication

Therapist Portal Clinical

  • Assign exercise protocols per patient
  • Set angle thresholds and rep targets
  • View patient session recordings & reports
  • Alert configuration for high injury risk
  • Export session data as PDF/CSV
  • Multi-patient management dashboard

Non-functional NFR

  • Inference latency <100ms per frame
  • WebSocket round-trip <200ms
  • Support 50 concurrent sessions (MVP)
  • JWT-secured API endpoints
  • HTTPS + data encryption at rest
  • Mobile-responsive patient UI

REST & WebSocket API Endpoints

Auth

POST/auth/googleExchange OAuth code → JWT token
POST/auth/refreshRefresh access token

Sessions

POST/sessions/startCreate session, return session_id
PUT/sessions/{id}/endFinalise session, compute summary
GET/sessions/{id}Fetch session detail + error log
GET/sessions?user_id=&page=Paginated session history

Real-time Inference

WS/ws/session/{session_id}Stream pose frames → receive predictions + feedback

Analytics

GET/analytics/rom/{user_id}ROM trend over time per joint
GET/analytics/errors/{user_id}Error frequency breakdown
GET/analytics/progress/{user_id}Accuracy trend, rep quality score

Therapist

POST/protocolsCreate exercise protocol for patient
PUT/protocols/{id}Update thresholds / reps / sets
GET/therapist/{id}/patientsList assigned patients with latest session
GET/reports/{session_id}/pdfGenerate downloadable PDF report
